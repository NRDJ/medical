**Endpoints de prueba**

**Pacientes**

**Todos los pacientes:** http://localhost:8082/api/patients

**Visitas**

**Por ID de Paciente:** http://localhost:8082/api/visits/by-patient/1

**Por rango de fecha:** http://localhost:8082/api/visits?from=2025-08-01&to=2025-09-01
